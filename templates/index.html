Examples to routes

  • List all peoples (GET) -> http://127.0.0.1:5000/peoples
  • List peoples per region (GET) -> http://127.0.0.1:5000/peoples/sudeste *see the list of regions
  • List all regions -> http://127.0.0.1:5000/regions

    • Paginate

    All routes are paged by default in 'page'=1 and 'page_size' = 20
    Example of the paged people route:
  • List all peoples paginated (GET) -> http://127.0.0.1:5000/peoples?page_size=10&page=2
